# Dynamic Color Clock (React + npm)

This project is a simple dynamic clock built using **React** and **Vite**.  
It displays the current date and time using the `date-fns` library for formatting and includes basic CSS styling with colors beyond black and white.

The project was created as part of a lab to practice React fundamentals, JSX, npm dependency management, and basic styling.

---

## Project Objectives

- Set up a React project using Vite
- Use JSX to render UI components
- Use the `date-fns` npm package to format date and time
- Display the current date and time on the page
- Apply CSS styling with additional colors
- Run and test the application locally
- Document the project clearly for maintainability

---

## Technologies Used

- React
- Vite
- JavaScript
- npm
- date-fns
- CSS
